System Backup & Restore

Manage your system data backups and restoration

Important Warning

System backup and restore operations are critical. Please read the following carefully:

  • Backup files contain all sensitive data including passwords and financial records
  • Store backup files in a secure location with encryption
  • Restoring a backup will completely replace all current data
  • Always test backup files before relying on them
  • Only SUPER_ADMIN users can perform backup/restore operations

Security Recommendations

  • Store backup files in encrypted storage
  • Test backup files regularly to ensure they work
  • Keep multiple backup copies in different locations
  • Use strong passwords for backup file access
  • Document backup procedures and recovery steps
  • Train authorized personnel on backup/restore procedures
