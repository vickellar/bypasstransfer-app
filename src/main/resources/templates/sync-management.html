How Sync Works
For Offline Users:
  • Record transactions using the "Offline Transactions" section
  • Transactions are stored locally until internet connection is available
  • System automatically attempts to sync when online
  • All data is preserved even if device goes offline
For Admin Users:
  • Monitor sync status through this dashboard
  • Manually trigger sync for all pending transactions
  • Retry failed sync operations
  • View detailed sync history and statistics
Tip: Sync operations are automatic but can be triggered manually for immediate processing.
